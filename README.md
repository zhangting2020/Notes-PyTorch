# Notes-PyTorch
PyTorch 0.4.0学习笔记是我于2018年5月3日开始记录的，参考了[PyTorch官网](https://pytorch.org/tutorials/)的英文文档，
以及[PyTorch0.3.0中文文档](http://pytorch.apachecn.org/cn/docs/0.3.0/index.html)。为什么自己要再写一个中文文档呢？
因为我觉得最好的方式是阅读英文文档，然后自己理解后写笔记。并且因为参考的中文文档是早前的版本，目前最新的PyTorch有
许多改动。

## 准备工作
- 安装PyTorch 0.4.0，使用的是Linux环境，按照官网的[提示](https://pytorch.org/)，使用Conda安装
- 安装Jupyter Notebook

## 学习流程
这里我个人是学了[官网教程](https://pytorch.org/tutorials/)的前5个部分，详细见笔记Part1~Part5。
但后来我感觉其实可以先学习”60分钟快速入门“，然后开始看[PyTorch文档](https://pytorch.org/docs/stable/index.html)就可以。

我的研究领域是计算机视觉，因此笔记目前不涉及其他内容的笔记。可以按照下面列出的内容进行学习，当然也可以看一看Part2~Part5部分，
因为这些是概要性的学习文档。

- 60分钟快速入门
- PyTorch文档
